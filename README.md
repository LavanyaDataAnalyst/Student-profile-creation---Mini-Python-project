# 🐍 Basic Student Profile Creation Program
## ℹ️ Description:
* The program collects user details such as name, age, and favourite subjects and displays them in a structured self-introduction format.  
* It demonstrates fundamental Python concepts like input handling, string manipulation, and list operations.

## 📖 Table of Contents: 
>* Project Overview
>* Features
>* Tools & Technologies
>* Sample output
>* How to Use


## 🔍 Project Overview

* This project is a simple Python-based application that collects user input and generates a formatted self-introduction.  
* It also allows users to update their favourite subjects and view the updated output.  
* It helps beginners understand input handling, string formatting, and basic Python operations.  

## ⚙️ Features

* Accepts user input dynamically  
* Splits and stores multiple subjects using lists  
* Allows updating of favourite subjects  
* Displays output in a structured format  
* Beginner-friendly Python logic  

## 🛠️ Tools & Technologies

* Python – Core programming language  
* Jupyter Notebook / VS Code – Development environment  


## 💡 Sample Output

```
Student Profile

Name: Lavanya
Age: 27
Favourite subjects: Excel, PowerBI

Updated Profile

add your favourite subjects: SQL, Python
Updated subjects: SQL, Python, Excel, PowerBI

```

## ▶️ How to Use

* 📜 How to Run
* Install Python on your system
* Open the .ipynb file in Jupyter Notebook or VS Code
* Run all cells
* Enter the required details when prompted
* View the generated student profile output
